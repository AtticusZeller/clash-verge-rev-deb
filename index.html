DEB Package Repository

This is a custom APT repository hosting various Debian packages.

How to Use

Add this repository to your system:

echo "deb [trusted=yes] https://atticuszeller.github.io/deb-index stable main" | \
    sudo tee /etc/apt/sources.list.d/atticuszeller-deb-index.list

Update package index:

sudo apt update

Install packages:

sudo apt install clash-verge

Available Packages

  • clash-verge-rev

Repository Structure

  • AMD64 Packages
  • ARM64 Packages
